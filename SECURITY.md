# Security Policy

## Supported Versions

Only the `main` branch, the current `latest` image tag, and the current upstream-aligned version tags are supported with security fixes.

| Version                       | Supported |
| ----------------------------- | --------- |
| main                          | yes       |
| latest                        | yes       |
| current upstream-aligned tags | yes       |
| older                         | no        |

## Reporting a Vulnerability

Do not open public issues for suspected vulnerabilities.

- Preferred: GitHub private vulnerability reporting for this repository via the Security tab
- Fallback: email `security@aethereal.dev`

Please include:

- affected repo, branch, or image tag
- reproduction steps
- impact assessment
- any confirmed mitigation
- whether the issue is packaging-specific, upstream-app-specific, or not yet clear

Please do not include live secrets in reports, and do not test against systems you do not own or have permission to assess.
